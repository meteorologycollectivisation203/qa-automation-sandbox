## 🚀 What this app does

qa-automation-sandbox is a free sandbox for QA automation practice on Windows. It gives you a full-stack app with a UI, an API, and a database in one place. You can use it to test sign in flows, user actions, data checks, and API requests without setting up your own server.

It includes:

- A social network style web app
- A built-in API with 65 endpoints
- A database for persistent test data
- 8 test users for login and role checks
- 55+ test cases for practice
- One command to launch the full stack

## 🖥️ What you need

## 🪟 Install on Windows

### 1. Save the project
Download the project from the link above and place it in a folder you can find again, such as:

- `Downloads`
- `Desktop`
- `Documents`

### 2. Extract the files
If the download is a zip file:

- Right-click the file
- Choose Extract All
- Pick a folder
- Finish the extraction

### 3. Open the folder
Open the extracted folder and look for the main launch file or script. The project is designed to start with one command, so you should see a clear start path in the folder.

### 4. Run the app
Double-click the Windows start file or open a terminal in the folder and run the start command included with the project. The app will start the UI, API, and database together.

### 5. Open the site
After the app starts, open your browser and go to the local address shown in the terminal or launch window. This is usually something like:
